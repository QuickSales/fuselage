## Install

<!--install-->

Add `@quickchat.vn/fuselage-tokens` as a dependency:

```sh
npm i @quickchat.vn/fuselage-tokens

# or, if you are using yarn:

yarn add @quickchat.vn/fuselage-tokens
```

<!--/install-->
